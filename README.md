# CPMNuker
Car Parking Multiplayer Hacking ToolKit

# requirements:
- `cmake` and `make`
- `gcc`

# installation:
- `git clone https://github.com/CPMNuker/CPMNuker.git`
- `cd CPMNuker`
- `mkdir build && cd build`
- `cmake ..`
- `make`

# update:
- `cp CPMNuker`
- `rm -rf build`
- `git pull`
- `mkdir .build && cd build`
- `cmake ..`
- `make`
